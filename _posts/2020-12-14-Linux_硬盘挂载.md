## 挂载
```
# 查看可挂载的磁盘
fdisk -l

# 查看已经挂载的磁盘
df -h

# 列出所有可用块设备的信息
lsblk

# 初始化磁盘
mkfs.ext4 /dev/vdb

# mount 磁盘到/home，保证/home为空
mount /dev/vdb /home

# 获取磁盘的uuid和属性，用uuid来进行开机mount
blkid

# 开机自动挂载
echo "UUID=xxx /home ext4 defaults 0 0" >> /etc/fstab
```
